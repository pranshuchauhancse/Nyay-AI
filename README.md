# NayahAI
NyayAI is an AI-powered LegalTech platform for India that helps users understand legal issues using Indian laws (IPC, CrPC, CPC, Contract Act, TPA). Built with FastAPI, OpenAI, LangChain, PostgreSQL + pgVector, and React, it provides legal Q&amp;A, statute mapping, case references, document drafts, and a freemium model for scalable access to justice.
